Banco de Dados

banco de dados é uma coleção organizada de informações - ou dados - estruturadas, normalmente armazenadas eletronicamente em um sistema de computador. Um banco de dados é geralmente controlado por um sistema de gerenciamento de banco de dados (DBMS)Um

Quem nos leciona nessa materia é o Professor Iury, que nos avalia da seguinte forma:

  • Atividades passadas em sala ou no teams;
  • Prova.

Atividades

Durante as aulas, o professor nos passa diversos exercicios para que em dupla, desenvolvavamos diagrama um banco de dados no app Dia Portable, ou algum outro de nossa escolha. Após isso, ele abre uma atividade no teams para que possamos postar esses diagramas lá, isso quando ele não passsa o que deve ser feito no próprio app.

Provas

A prova geralmente é feita em dois dias, o segundo sendo para aqueles que perderam a prova no primeiro dia. Na prova caem os conteudos que o professor foi nos passado e explicado durante todo o bimestre, sendo disponilizados também os slides utilizados pelo professor no Teams.

Atividades desenvolvidas em sala

Abaixo estão fotos de algumas atividades que desenvolvevamos as aulas.

Uma Auto Escola precisa gerenciar as tarefas de aulas e para isso será necessário a criação de um banco de dados.

Uma empresa organizadora de eventos esportivos deseja informatizar o controle da organização de um campeonato de futebol amador, e para isso será necessário a criação de um banco de dados.

Uma farmácia necessita de um banco de dados para auxiliar no gerenciamento das vendas, controle de estoque e clientes.

Uma auto mecânica solicitou a criação de um banco de dados para controlar o fluxo dos serviços prestados pela empresa. O proprietário necessita controlar os serviços, veículos e clientes.
